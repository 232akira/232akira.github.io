Hexo初始化博客

基于Node.js的博客框架。本博客采用Hexo+Github搭建。
常用命令

功能 命令 简写
清理缓存 hexo clean hexo cl
部署站点 hexo deploy hexo d
生成博客 hexo generate hexo g
启动本地服务 hexo server hexo s
